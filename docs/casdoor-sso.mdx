## Introduction
Casdoor is a UI-first centralized authentication / Single-Sign-On (SSO) platform based on OAuth 2.0 / OIDC. It provides OAuth 2.0 / OIDC based signup and login, as well as third-party login including GitHub, Google, QQ, WeChat, etc.
CasWAF uses Casdoor as an SSO provider to manage users and permissions.

## Login to CasWAF
CasWAF supports logging in with Casdoor. You need to register CasWAF as a Casdoor application first.

About how to register CasWAF as a Casdoor application, please refer to [Casdoor documentation](https://casdoor.org/docs/application/config).

After registering CasWAF as a Casdoor application, you can log in to CasWAF with Casdoor.

![login](/img/casdoor-sso/login.png)

Once you login. You can manage sites and certs as [Sites documentation](/docs/site) and [Cert documentation](/docs/cert) mentioned.
